OS File Directory Structure Simulation

Single-Level Directory Structure

A single-level directory system contains all files in one directory. It's simple but limiting, as all filenames must be unique and organization is difficult with many files.

Hashed Directory Structure

A hashed directory uses a hash function to determine the location of files, enabling faster lookups. Files are distributed into buckets based on their hash value.

Indexed Directory Structure

An indexed directory structure maintains an index table with pointers to file locations. This provides fast lookups and allows for more flexible storage structures.

Hierarchical Directory Structure

A hierarchical directory structure organizes files in a tree-like structure with directories and subdirectories. This allows for better organization and file management.
